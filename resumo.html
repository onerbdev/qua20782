Aprendendo mais sobre Git e GitHub

Abaixo estarão disponíveis informações básicas de como usar essas ferramentas para otimizar o seu processo de desenvolvimento.

Quais os significados e utilidades dessas ferramentas ?

  • Git

    • O git é uma ferramenta de versionamento de código, de forma local. Ele se baseia em rastrear alterações/o que especificamente foi alterado.

  • GitHub

    • O github é uma ferramenta de hospedagem online para projetos e permite o desenvolvimento colaborativo.

Comandos básicos

  • git config user.name "nome-usuario"

    • Esse comando é usado para configurar o usuário

  • git config user.email "seu@email.com"

    • Esse comando é usado para configurar seu email
