COMANDOS BÁSICOS GIT

  • Configuração inicial

    • Esses comandos são usados uma vez para configurar seu nome e email no git:

    git config --global user.name "Seu nome"

    git config --global user.email "Seu email"

  • Criar ou clonar repositórios

    • Iniciar um repositório local

  • git init
  • git clone URL_del_repositorio
  • git status
  • git add file
  • git add .
  • git commit -m "Mensagem da alteração feita"
  • git branch
  • git branch nome_ramo
  • git checkout nome_ramo
  • git checkout -b nome_ramo
  • git remote -v
  • git push origin nome_ramo
  • git pull origin nome_ramo
  • git log
  • git diff
  • git reset file
